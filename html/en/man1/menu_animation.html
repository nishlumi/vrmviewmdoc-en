6.13.1.4. Animation tabÔÉÅ


This tab has buttons for functions related to animation.

In Project

Create New:

Create a new animation project.

Open:

Open an existing animation project file.

Save:

Save the current animation project to a file.

Config:

Open the animation project settings screen.

Note

You can save with a name and save or overwrite.

In the Play operation

play from beginning:

Play the animation from the 1st frame.

Play/pause from the middle:

Play or pause animation.

Stop:

Stop the animation completely.

Loop:

Toggles loop playback on/off.

  • Moved default duration to Project setting.

In Frame operation

the current:

This is the currently selected frame number. If you change it, it will move to that position and apply the pose.

maximum:

Change the maximum number of frames. Note that if you decrease it, the key at that position will be deleted.

Apply:

Actually apply the modified maximum number of frames.

In Key operation

Register keyframe‚Ä¶:

ver 2.1.0 or later Displays the keyframe registration window. The window that appears can be opened by right-clicking anywhere within a tab on the ribbon bar.

Cut/Copy/Paste:

Manipulate keyframes of the current object.

Delete Keyframe‚Ä¶:

Deletes keyframes on the currently selected timeline. Please note that it is not the selected 3D object.

Contents that can be deleted:

  • the keyframe itself

  • properties only

By deleting only the properties, the keyframes will only contain common properties such as translation and rotation.
In addition, when only the property is deleted, the keyframe itself is not deleted yet. Choose Remove keyframe to completely remove it from the timeline.
