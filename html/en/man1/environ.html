3. Operating environment

OS:

Windows, macOS, Linux, ChromeOS [1], Android(including VR devices)

Supported browser:
Edge, Chrome, Opera, Vivaldi, Firefox [2] , Browser of VR devices
Ttraffic:
First boot: about 20-25MB
Stage change/effect usage: 30MB (first use only)
Immediately after update: 1KB～25MB
Normal for the second and subsequent times: about 1 to 10 KB
Web application version only
Memory:

4GB or more of PC memory is recommended.

  • You can adjust the usage within the app within the range of 256MB to 4GB

  • Developed on a PC with 8GB memory

Storage:

The more the better. [3]

Graphics performance:

Checked with the following PC

  • PC with GTX 1060 to RTX3060 Ti

  • Last Intel CPU model MacBook Air to M1 MacBook Air

  • SUS UX21A notebook PC

  • Meta Quest 3, PICO4

Access to the outside:

Google, jsdelivr, unpkg

  • Purpose of using style sheets and javascript libraries

  • 上記以外への外部へのアクセスは行っていません。

  • ユーザーデータを外部へ送信する機能は一切持っていません。

Access to terminal (PC):
<Open>
・VRMなど3Dオブジェクトファイル、画像ファイル、音楽ファイル
・動画ファイル
・.vvmposeファイル、.vvmmotファイル、.vvmprojファイル

<Save>
・スクリーンショット
・動画ファイル
・.vvmposeファイル、.vvmmotファイル、.vvmprojファイル
[1]

You can’t use the PC version on ChromeOS.

[2]

Firefox cannot do independent windowing of PWAs.

[3]

Some files such as pose files and project files are saved in the app’s internal storage, but they are all saved inside the PC. It is not saved externally.

Note

  • Android and iOS are not supported according to Unity’s WebGL specifications. (Actually, you can use it if you access it, but it is difficult to operate)

  • The traffic volume is only a guideline. In principle, communication is only performed when loading or updating the application because it is PWA compatible.

  • Unity itself and additional libraries are not required for use.

  • Since it supports PWA (Progressive Wab App), it can be installed and used independently from the browser.

  • Windows, macOS, and Linux also have PC versions that run independently.

  • Browsers/extensions with mouse gestures may result in incorrect operation. It is necessary to turn it off in advance or perform the key operation described later.

  • After the first time, do not reload more than necessary. When an update is ready, we will notify you on the app.
