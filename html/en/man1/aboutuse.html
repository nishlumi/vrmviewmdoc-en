2. Prerequisites for Use

This application is free of charge. No user registration required, which is often the case with web apps. I myself dislike web apps that claim to be free and ready to use, but require user registration to use! I myself dislike web apps that claim to be free and ready to use, but require user registration for every single use, so I aimed for this app to be really ready to use as soon as you access it, just like a smartphone app or PC software. (I guess it’s necessary for marketing etc., so it can’t be helped.)

The web-app version may have delays depending on the server on which it runs. Also, of course, an Internet connection is required. If you want to use it in a stable environment without communication, we recommend you to use the PC version described below.

The website “VRM Township Portal” to which this application is published and linked uses Google Analytics to determine the number of people accessing the site. Please be aware of this.

This application itself does not have any function to manage user information on the server or access external services to exchange data. Therefore, we are not able to directly grasp the number of users, so if you use this application, we would be very happy if you could appeal that you are using it on each social networking service.


2.1. Restrictions

Please read the following instructions carefully before use. Because the base Unity build is based on the WebGL method, it is more dependent on memory and graphics performance [1] than native OS applications. It cannot withstand the production of animations that are too long and use a lot of 3D objects.

  • More high quality & beautiful photos!

  • More lengthy and generic animations!

In these cases, we recommend using other applications that have native versions for each OS. For those who are more familiar with the OS, it is best to use the Unity editor, Blender, etc. for the final version.

[1]

In particular, shader-related items are greatly restricted. note that.
